EMS-ESP Installer

Install the latest firmware onto a BBQKees Electronics EMS Gateway board 

BBQKees Electronics' EMS Gateways boards, and equivalent, can be flashed with the latest EMS-ESP firmware when connected directly to your computer. Any EMS-ESP configuration settings and stored data are not erased.

EMS-ESP firmware version is the most current version. Release Notes.

Note: This latest stable version is recommended for most users. You can always decide later to upgrade to the development releases from the EMS-ESP WebUI.

To use, attach the Gateway device directly to your computer using a USB cable, choose the model below and then click →
