EMS-ESP Installer

Install the latest firmware onto a BBQKees Electronics EMS Gateway board 

BBQKees Electronics' EMS Gateways boards can be installed with the latest EMS-ESP firmware when connected directly to the USB port of your computer. Connect.

To use, select the Gateway model below, plug in a USB-Cable and click Connect.

If you are unsure which version of the EMS-ESP firmware is currently installed, click on the Connect button, choose "Open Console" (if prompted press CTRL-C) and type show system.

Note: The firmware is also compatible with any ESP32's that match the same chip type (ESP32S or ESP32-S3) and memory profile (16MB flash with additional PSRAM).
